# Michael Mugnai personal Portfolio
This is the repo corresponding to my personal website, created with Jekyll and hosted by Github.

In order to edit it, you need to install Ruby and Jekyll, then follow the steps below.


## Clone and run
It's really simple:

``ssh
git clone git@github.com:Maik93/Maik93.github.io.git
cd Maik93.github.io
jekyll serve --force_polling
``
